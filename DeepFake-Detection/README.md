# Explainable DeepFake Detection

DeepFake Detection using:
- EfficientNetB0
- CBAM Attention
- Grad-CAM
- PyTorch

## Features
- Fake vs Real face classification
- Attention-enhanced CNN
- Explainable AI visualization
- GPU training in Google Colab

## Dataset
https://www.kaggle.com/datasets/ciplab/real-and-fake-face-detection

## Future Improvements
